About Blockpress.me

Blockpress.me allows the user to create highly customisable and themable websites, that can make use of ajax loading content modules to pull data from multiple javascript apis.

Steem posts

Our first main content module, which uses the Steem blockchain API, will allow the webhost to create any site that needs posts and comments, such as blogs and forums. We use this module to display our news feed.

Future Developement

We are currently working towards our initial beta release. Keep track of our progress on github.
