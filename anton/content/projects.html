Open source projects

Anton has built and maintained a number of FLOSS projects over the years, including serveral wordpress plugins. Their most recent projects include the SteemSwitch firefox plugin and the BlockPress frontend CMS.

Current projects

BlockPress

Anton is developing BlockPress.me with Dana. BlockPress.me is a FLOSS frontend CMS that uses blockchain content modules as its data. BlockPress powers this site and its own website, and is nearing first beta release!

SteemSwitch

SteemSwitch is a Firefox browser plugin that allows the user to switch easily between steemit, busy, utopian, steemd, steemdb and other sites based on the steem blockchain.

Anton has a plan to port this code to the Chrome browser at some point in the near future.

Old projects

Wordpress plugins

Anton developed several wordpress plugins, although these are no longer maintained. If anyone wishes to fork and maintian them, they should feel welcome to do so.

bbPress2 shortcode whitelist

bbPress2 shortcode whitelist was built to allow a limited selection of wordpress shortcodes to be allowed in comments. It automatically detected certain plugins, such as Anton's own bbPress2 BBCode plugin, and his Video and Audio BBcode plugin, to allow them to be used. All other wordpress shortcodes are only allowed by the site authors.

bbPress2 BBCode

bbPress2 BBCode was developed in order to add support to forum style BBCodes to wordpress posts and comments. Originally he built it for the bbPress2 forum extension project, hence the name, but he soon expanded it to work in regular wordpress posts, comments and BuddyPress user profile fields.

Video and Audio BBCodes

Video and Audio BBCode added support for special video shortcodes for embedding youtube and other videos in WordPress posts and comments. It became redundant when WordPress starting embedding them automatically with just a link.

Custom profile filters for BuddyPress

Anton developed Custom profile filters for BuddyPress specifically extended BuddyPress user profile pages to allow special fields to be added. If installed in combination with bbPress2 BBCode, it allowed BBCode formatting to be used in profile fields. It also gave users control over which words turned into search fields, and allowed special social media link fields for twitter to be added.

Elgg plugins

Before WordPress, Anton wrote a few plugins for Elgg. Their final plugin was to export all the Elgg data to a wordpress export format XML file. This enabled an easy transition from Elgg to BuddyPress.

EnigMagick

EnigMagick is a simple tool for calculating a numeric value for a given string of text and finding words and phrases with a matching value in another piece of text. Themeable. Whilst not very useful in its current form, I had an idea to use it to extend to disguise secret messages as strange prose. Only those with the correct 'key' text would be able to decode it correctly.

ChaosHex

ChaosHex is my only old project not written in PHP. This one was instead written using BASH scripts. It is a frivolous demo inspired by ideas like those found in the Martix, that we live inside a computer simulation, and can hack the program from within. It was really an excuse to make pretty patterns appear on the screen.
