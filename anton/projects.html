Open source projects

Anton has built and maintained a number of open source projects over the years, including serveral wordpress plugins. Their most recent projects include the SteemSwitch firefox plugin and the BlockPress frontend CMS.

SteemSwitch

SteemSwitch is a Firefox browser plugin that allows the user to switch easily between steemit, busy, chainbb, utopian, steemd, steemdb and other sites based on the steem blockchain.

Anton has a plan to port this code to the Chrome browser at some point in the near future.

BlockPress

Anton designed and is building the BlockPress.me a FLOSS frontend CMS that uses blockchain content modules as its data. Although currently in alpha BlockPress powers this site and its own website!
